DealHopper

DealHopper is a smartphone price comparator designed to help users find the best deals across multiple online stores, such as MediaMarkt, PhoneHouse and Backmarket. With its intuitive interface and powerful features, DealHopper simplifies the process of price comparison and recommendation.

Technologies

DealHopper is built using the following tools and frameworks:

  • Django: Framework for managing the web application.
  • BeautifulSoup: For scraping smartphone prices and details from online stores.
  • Whoosh: For implementing an efficient search index.
  • Recommendation System (RS): Provides personalized recommendations based on user preferences.

Important Information

Some online stores use lazy loading for their content, which makes tools like BeautifulSoup less effective. An alternative approach would be to use Selenium or other interface automation tools to handle dynamically loaded content. However, this project prioritizes learning, and the use of BeautifulSoup is intentional for educational purposes.

Other stores feature dynamic pricing that varies over time, making direct comparisons more challenging. Despite this, they are included due to the limited availability of alternative options. Also, there are shops, such as PcComponentes and ElCorteInglés, which return a 403 Forbidden response when accessed via Python's HTTP libraries.
